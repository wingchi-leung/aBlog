### 简介

基于Springboot+thymeleaf+mybaits搭建的个人博客系统，用于个人学习交流。

更新中 

### 技术栈

后端：

- SpringBoot
- SpringSecurity
- Mybatis
- mysql

前端：

- html,js,css
- thymeleaf 
- editor.md

### 快速运行

1. 克隆项目到本地 
2. 根据本地情况，修改项目中数据库的配置，配置文件位置为`src/main/resources/application.properties`
3. 将resources目录下的blog.sql文件在Mysql数据库中执行。 
4. 配置完后，通过主程序即可启动。
